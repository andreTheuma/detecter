weave/3

weave(SrcDir, MfaSpec, Opts) -> [comp_ret()]

  • SrcDir = string()
  • MfaSpec = analyzer:mfa_spec()
  • Opts = options()

Instruments all modules in the specified directory by weaving in tracing and monitor instructions through a transformation of the corresponding module abstract syntax tree.

SrcDir
The directory path containing the source or object files to be inspected and weaved.
MfaSpec
The function that specifies which remote (i.e., external) function calls are to be weaved with tracing and monitor instructions.

Object (.beam) files that require instrumentation need to be compiled in debug mode.

The following options are available:

{outdir, Dir}
The directory where the generated weaved files should be written. Defaults to the current directory ..
{i, Dir}
The directory containing include files that the source files in SrcDir depend on.
{filer, Fun}
The filter function that suppressed events. Defaults to allows all.
erl
Instructs the compiler to output the generated files as Erlang source code rather than beam. Defaults to beam.

Returns: A list containing the compilation result of each file in SrcDir. Warnings are included if present.
