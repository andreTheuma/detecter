compile/5

compile(Mod, LexerMod, ParserMod, File, Opts) -> any()

Compiles the specified file containing one or more properties specified in MaxHML, and writes the executable monitor descriptions to file.

Mod
Module implementing the callback function visit/2.
LexerMod
Module that performs the lexical analysis of MaxHML scripts.
ParserMod
Module that parses the lexer tokens to return the MaxHML syntax tree.
File
Path where the file to compile resides.
Opts
Compiler options.

The following options are available:

{outdir, Dir}
The directory where the generated output monitor file should be written. Defaults to the current directory ..
erl
Instructs the compiler to output the generated monitor as Erlang source code rather than beam. Defaults to beam.

Errors and warnings that arise during the compilation are reported on the standard output. No monitor file is output unless compilation completes with no errors. The output depends on whether the flag erl is specified or otherwise.

Returns: ok if compilation succeeds, {error, Reason} otherwise.
