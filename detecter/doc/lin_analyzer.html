analyze/3

analyze(Event, M, PdList) -> {[pd()], monitor()}

  • Event = event:evm_event()
  • M = monitor()
  • PdList = [pd()]

Analyzes the specified trace event and performs the monitor reduction accordingly.

Event
Trace event to analyze.
M
Monitor to analyze the trace event.
PdList
Initial list of proof derivations. New derivations are added to the head of the list.

The reductions effected are added to the head of the specified proof derivation list. The function assumes that the monitor supplied is already in the ready state. See lin_analyzer:reduce_tau/2.

Returns: Proof derivation list and reduced monitor. The specified monitor is guaranteed to be reduced by at one or more steps, where the first analyzes the specified trace event.

dispatch/1

dispatch(Event::event:int_event()) -> term()

Dispatches the specified abstract event to the monitor for analysis.

Event
The abstract event that the monitor is to analyze.

Returns: Depends on the event type. See event:event/0.

  • When event is of type fork, the PID of the new child process is returned;
  • When event is of type init, the PID of the parent process is returned;
  • When event is of type exit, the exit reason is returned;
  • When event is of type send, the message is returned;
  • When event is of type recv, the message is returned.

reduce_tau/2

reduce_tau(M::monitor(), PdList::[pd()]) -> {[pd()], monitor()}

Reduces the specified monitor using tau transitions repeatedly until it cannot be reduced further.

M
Monitor to reduce.
PdList
Initial list of proof derivations. New derivations are added to the head of the list.

The monitor is left in a state that is ready to analyze trace events.

Returns: Proof derivation list and reduced monitor. If the specified monitor cannot be reduced, the original monitor is returned, and the proof derivation list is empty.
